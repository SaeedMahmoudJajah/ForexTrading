## A. The master workflow (memorize this)

```
STRUCTURE → LEVEL → CONFLUENCE → TRIGGER → RISK → MANAGE → JOURNAL
```
Read trend → find the level → stack independent reasons → wait for the candle/CHoCH trigger
→ define stop & target (≥2R, ≤1% risk) → manage (break-even, trail, scale) → log it.

## B. Candlestick signals (Ch.2)

| Signal | Shape | At a... | Bias |
|--------|-------|---------|------|
| Hammer / Bullish pin | Small body, long lower wick | downtrend low / support | Bullish reversal |
| Shooting star / Bearish pin | Small body, long upper wick | uptrend high / resistance | Bearish reversal |
| Bullish engulfing | Up-body swallows prior down-body | support | Bullish reversal |
| Bearish engulfing | Down-body swallows prior up-body | resistance | Bearish reversal |
| Morning star | Down → small star → up | bottom | Bullish reversal |
| Evening star | Up → small star → down | top | Bearish reversal |
| Doji / Spinning top | Tiny body | extremes | Indecision / possible turn |
| 3 White Soldiers / Black Crows | 3 strong same-way candles | trend | Strong momentum |

**Rule:** a candle is a *trigger* only at a meaningful level, with-trend, on a closed candle.

## C. Market structure (Ch.3)

```
UPTREND   = Higher Highs + Higher Lows (HH/HL)   → look for LONGS on pullbacks to HL
DOWNTREND = Lower Highs + Lower Lows (LH/LL)      → look for SHORTS on pullbacks to LH
RANGE     = flat highs & lows                     → trade edges or stand aside

BOS  (Break of Structure)   = break in trend direction  → trend CONTINUES
CHoCH(Change of Character)  = first break against trend  → trend may be TURNING

False break / liquidity grab = wick beyond a level that closes back inside → fade it.
Stops go BEYOND structure with a buffer, never on the obvious level.
```

---

## D. Support, resistance & zones (Ch.4)

- **Support** = floor (demand). **Resistance** = ceiling (supply).
- **Polarity:** broken resistance → support; broken support → resistance (best retest entries).
- Think **ZONES, not lines** (bodies = core, wicks = edges).
- **Supply/demand zone** = the base a strong impulsive move launched from; fresh = cleanest.
- Strength ↑ with: more touches, stronger reaction, higher timeframe, and confluence.
- Three plays: **bounce**, **breakout**, **retest** (always + trend + trigger + risk).

## E. Chart patterns (Ch.5)

| Pattern | Type | Bias / break |
|---------|------|--------------|
| Head & Shoulders / Inverse | Reversal | Break of neckline |
| Double / Triple Top | Reversal | Down on level break |
| Double / Triple Bottom | Reversal | Up on level break |
| Bull / Bear Flag, Pennant | Continuation | With the trend |
| Ascending triangle | Continuation/breakout | Usually up |
| Descending triangle | Continuation/breakout | Usually down |
| Symmetrical triangle | Neutral | Trade the breakout |
| Rising wedge | Reversal/cont. | Usually down |
| Falling wedge | Reversal/cont. | Usually up |
| Cup & Handle | Continuation | Up |

**Measured move target:** project the pattern's height (or flagpole) from the breakout.
**Always:** require a *close* beyond the boundary; consider entering the retest; R/R ≥ 2:1.

## F. Indicators (Ch.6) — starter toolkit

| Tool | Family | Use | Key setting |
|------|--------|-----|-------------|
| EMA 50 / 200 | Trend | Bias + dynamic S/R + crosses | 50, 200 |
| RSI | Momentum | **Divergence** + 50-line filter | 14 |
| ATR | Volatility | **Volatility-based stops** (e.g. 1.5× ATR) | 14 |
| ADX | Trend strength | Trend (>25) vs range (<20) filter | 14 |

```
• Indicators LAG and CONFIRM — they never lead price.
• "Overbought/oversold" ≠ reversal in a trend (price can walk the band / stay >70).
• 1 trend tool + 1 momentum tool MAX. Don't stack redundant oscillators.
• Divergence = momentum warning → confirm with structure/level before acting.
```

## G. Fibonacci (Ch.7)

```
RETRACEMENTS (pullback entries):  23.6% · 38.2% · 50% · 61.8%⭐ · 78.6%
   "Golden pocket" = 0.618–0.65 → prime with-trend pullback entry zone.
EXTENSIONS (targets):             127.2% · 161.8%⭐ · 261.8%
Draw across a clean impulsive swing. Best when it overlaps real S/R / structure.
```

## H. Confluence (Ch.7) — the quality filter

```
Stack INDEPENDENT reasons at one price + direction:
  structure ▢  S/R or supply/demand zone ▢  polarity retest ▢  pattern ▢
  Fib golden pocket ▢  key MA ▢  round number ▢  candle trigger ▢
  RSI divergence ▢  HTF agreement ▢

3–4 independent confluences at a meaningful level = a STRONG setup.
(3 momentum indicators agreeing = 1 reason, not 3. Stack DIFFERENT things.)
```

## I. The pre-trade checklist (Ch.8) — tick all before entering

```
□ News checked (economic calendar) — no imminent high-impact surprise
□ HTF bias clear (trend/structure) — trade is WITH it
□ Price at a meaningful LEVEL / zone
□ Confluence stacked (3–4+ independent reasons)
□ TRIGGER present (closed reversal candle / CHoCH / reclaim)
□ Stop beyond invalidation (ATR/structure + buffer); target ≥ 2× risk
□ Position sized so the stop = ≤1% of account
□ Calm, not FOMO/revenge → execute with SL + TP, then JOURNAL it

Any box unticked → NO TRADE. The best trade is often none.
```

## J. Top mistakes to avoid

```
✗ Trading candles/patterns mid-range with no level or trend
✗ Fighting a strong higher-timeframe trend on a small reversal signal
✗ Front-running breakouts before a confirming close (false breaks)
✗ Stacking 8 indicators → contradictory noise & paralysis
✗ Treating "overbought" as an automatic sell in a trend
✗ Stops on the obvious level (gets liquidity-grabbed) — use a buffer
✗ No measured target / taking trades with reward < risk
✗ Skipping the journal — no journal, no improvement
```

> 🔑 Print this. The edge isn't any single item here — it's running the **whole workflow** with discipline, every time, and journaling the results until the data proves your setups have a positive edge.
